## Why

El objetivo de este change es formalizar y dejar documentada la estructura base inicial del repositorio Food Store usando el enfoque OPSX, para arrancar con una base organizada y modular.

## What Changes

- Crea las carpetas raíz: frontend/, backend/, docs/, openspec/
- Inicializa los archivos estructurales mínimos (.gitignore, README.md)
- Prepara la base para desacoplar el frontend y backend y escalar el desarrollo por módulos/líneas de negocio

## Capabilities

### New Capabilities
- `repo-structure`: Define la estructura, convenciones y organización base del proyecto

### Modified Capabilities
- (ninguna, es la primera estructura)

## Impact

- Cambios en la raíz del repo: organización de carpetas
- Inicialización de archivos base (README, .gitignore)
- Ningún API/producto afectado todavía. Solo estructura de proyecto.
